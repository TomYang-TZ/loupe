Comparison

Slide-over (A)

  • Log list stays visible — you can still see the stream
  • Click different entries to swap the detail without closing
  • Feels like VS Code / Linear sidebar

Modal (B)

  • Full focus on detail content
  • Familiar pattern (click → popup → close)
  • But blocks the log stream while open
