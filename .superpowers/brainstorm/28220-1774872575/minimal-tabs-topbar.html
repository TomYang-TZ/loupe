Tab Fix Details

Problem
  • Tab uses display:inline with text + × inline
  • No max-width on tab label — overflows container
  • Activity dot (position:absolute) collides with × button at top:4px; right:3px
  • No flex-shrink:0 on × button — gets squished
Fix
  • Tab becomes display:flex; align-items:center; gap:4px
  • Label gets max-width:80px + ellipsis truncation in minimal mode
  • × button gets fixed 14×14px size + flex-shrink:0
  • Activity dot repositioned to left of label (or removed in minimal — less noise)

Topbar Changes

  • Background elevated to --surface (#1e293b) with border — gives depth separation from --bg
  • Checkbox toggle → sun/moon SVG button (18px in topbar, 22px in full header)
  • Pin toggle slightly smaller (26×14px track) — proportional to compact layout
  • Consistent 8px gap between all controls
